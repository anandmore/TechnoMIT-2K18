Event Objectives:

The objective of these event is to test knowledge and some innovativeness in C programming .The event has been designed for people who love coding and who are passionate about programming. This encourages participants to test their coding skills and help them realize their current level of skill in programming.

1. Hash Coder

Round 1: Crossword puzzle

  • This round will consist of 50 points ,In which each participants will be given online crossword puzzle.
  • Time limit for this Round will be 20-25 minutes.
  • In the given time limit , we expect a considerable output from the participant's side .
  • Participant must be able to achieve maximum points ,to secure further position in the Competition.

Round 2: Programming

  • This Round will be the Final Stage for the Contest !
  • This Round will consist of 50 points.
  • In these round participant will be given two problem statement with half of the code and output of that code, Participants have to fill the blank spaces given in the code according the output given to them.

2. Code Kshetra

Round 1: Quiz Relay

  • This round will consist of 30 question which is of 30 points i.e.(Each Question will carry 1 mark ).
  • In this one of the team member will be given time of 5 minutes to solve the questions and after 5 minutes another member of team will replace him /her and continue with the remaining questions of the quiz .(all three members will be given chance one by one)
  • There will be Cuttoff for passing each round.
  • Participant must be able to achieve maximum points ,to secure further position in the Competition.
  • Time limit for these round will be 20mins .
  • (And will be extended if any technical mismatch )

Round 2: C Programming Relay

  • In this round a Problem statement is given to each Shortlisted  team  and they have to write code for these.
  • In this round one of the team members will be given time limit of 3 minutes for writing half of the code for the given problem.
  • After 3 minutes group members will have to switch and  other member will continue the previously written code by his/her group member.
  • In the given time limit , we expect a considerable output from that participant's side.

Participant must be able to achieve maximum points ,to secure further position in the Competition.

Rules :

  • All the participants must bring their College Id Card
  • Participants must report half hour before the event start
  • Participants must bring receipt of the event for verification
  • Participants should maintain discipline while Attending the event
  • The decision of Judges will be final
  • After each round participants are short listed for the next round according to their score.
  • In case of Tie between teams the result will be given according to minimum time taken by the team.
